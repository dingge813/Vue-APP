# Vue-APP
使用方法：
#### 1.将该项目下载到本地，在终端中进入该文件夹输入npm install来下载项目所需要的组件
#### 2.在本地运行项目：npm run dev
#### 3.运行项目所需要的组件 npm run build
#### 4.运行e2e测试：npm run e2e
#### 5.运行所有测试：npm test
#### 最后就能看到该项目的效果辣！！
